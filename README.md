# 教务系统管理后台 java58

## 项目概述

教务系统管理后台是一个基于Javaweb技术栈开发的教育教学管理平台。该平台采用Spring Boot与Vue.js构建前后端，以MySQL为数据库管理系统，实现了高效、稳定的课程管理、自动排课等功能。

## 技术栈

- 前端：Vue.js
- 后端：Spring Boot
- 数据库：MySQL
- 排课算法：遗传算法

## 功能模块

- **课程管理**：教师和学生可在线管理课程信息。
- **自动排课**：采用遗传算法实现智能排课。
- **成绩管理**：支持查看和更新成绩。
- **作业管理**：可发布、提交和批改作业。
- **课程表查看**：支持查看课程时间表。

## 系统角色

- 教师：可管理课程、发布作业、录入成绩等。
- 学生：可查看课程表、提交作业、查看成绩等。

## 运行环境

- 服务器：推荐使用Java环境配置的Linux服务器。
- 数据库：MySQL数据库服务器。
- 前端：现代浏览器。

## 部署步骤

1. 准备Java运行环境。
2. 部署MySQL数据库，并根据提供的脚本初始化数据。
3. 部署后端服务，配置相关环境参数。
4. 部署前端至Web服务器。
